Design a **Workspace UI screen** for my AI-powered MSME platform “ChakraAI”.

⚠️ Important:

* Keep the UI **very simple and clean**
* Do NOT overcomplicate with too many sections
* Focus only on: **tender discovery, insights, and report generation**
* Follow existing design system (colors, typography, spacing)

---

## 🧭 Workspace Purpose

This workspace is used to:

* Automatically discover relevant tenders
* Show AI-generated insights
* Generate structured tender reports

---

## 🏢 Layout Structure

### 1. Top Header (Simple & Clean)

Include:

* Workspace Name
* Short Description
* Small member avatars (optional)
* “Invite” button (minimal)

---

## 📊 2. Main Content (Single-Page Experience)

⚠️ Avoid multiple tabs — keep everything in one unified dashboard.

---

### 🔍 Section 1: Tender Insights Feed (Core Section)

This is the **main focus of the workspace**.

Display as vertical cards:

Each card includes:

* Tender Title
* Source (e.g., Government / Portal)
* Short AI Summary (2–3 lines)
* Key tags:

  * Eligibility
  * Deadline
  * Budget (if available)

---

### Card Actions:

* “View Details”
* “Generate Report”

---

### Visual Design:

* Clean cards with soft shadow
* Left icon (document/tender)
* Highlight important tenders subtly

---

## 📄 3. Report Generation Section

Keep this **very simple**:

When user clicks “Generate Report”:

Show a panel or modal with:

* Report Title
* Generated Summary
* Key Sections:

  * Overview
  * Eligibility Criteria
  * Required Documents
  * Opportunity Analysis

---

### Actions:

* Download (PDF)
* Save Report

---

## ⚡ 4. Quick Action Bar (Top or Side)

Add 2–3 simple buttons:

* “Scan for New Tenders”
* “Generate All Insights”
* “Refresh Data”

---

## 🟢 5. Empty State (Very Important)

For new workspace:

Show:

* Icon/illustration
* Title: “No tenders found yet”
* Description: “Click below to start discovering relevant tenders”

---

### Button:

* “Scan for Tenders” (Primary CTA)

---

## 🔔 6. Minimal Notifications Inside Workspace

Optional small alerts:

* “3 new tenders found”
* “Report generated successfully”

Keep subtle (top banner or small badge)

---

## 🎨 Design Guidelines

* Keep UI:

  * Minimal
  * Focused
  * Not cluttered
* Use:

  * Rounded cards (12–16px)
  * Soft shadows
  * Clean spacing
* Avoid:

  * Too many tabs
  * Complex navigation
  * Overloaded analytics

---

## 🎯 Goal

Create a **simple, powerful workspace UI** that feels like:

* AI assistant for tender discovery
* Insight-driven dashboard
* Report generation tool

The user should be able to:

1. See tenders
2. Understand them quickly
3. Generate reports

All in one clean screen.

Do not overdesign. Keep it practical and industry-ready.
